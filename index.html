Command List

Moderation

  • /ban: Bans a user from the server
  • /kick: Kicks a user from the server
  • /mute: Mutes a user for a specified duration
  • /unmute: Unmutes a user
  • /clear: Clears a specified number of messages
  • /warn: Warns a user
  • /lock: Locks a channel, preventing messages
  • /unlock: Unlocks a channel, allowing messages
  • /slowmode: Sets slowmode for a channel
  • /role: Assigns or removes a role from a user

Utility

  • /ping: Replies with Pong! and latency
  • /serverinfo: Displays server information
  • /userinfo: Displays user information
  • /avatar: Displays a user’s avatar
  • /whois: Looks up a user by ID
  • /uptime: Shows the bot’s uptime

Fun

  • /roll: Rolls a dice
  • /coinflip: Flips a coin
  • /poll: Creates a poll
  • /say: Makes the bot say a message
  • /meme: Shares a random meme
  • /rps: Play rock-paper-scissors
  • /8ball: Ask the magic 8-ball a question
  • /joke: Tells a random joke

Information

  • /help: Lists all available commands by category
  • /botinfo: Displays bot information
  • /channelinfo: Displays channel information
  • /roleinfo: Displays role information
  • /invite: Generates an invite link for the bot
  • /stats: Shows server statistics

Economy

  • /balance: Check your or another user’s balance
  • /give: Give coins to another user
  • /shop: View available items in the shop
  • /buy: Buy an item from the shop
  • /leaderboard: Shows the top coin holders

Events

  • /eventcreate: Creates a server event
  • /eventlist: Lists upcoming server events
  • /eventjoin: Join a server event
  • /eventleave: Leave a server event

Frequently Asked Questions

How do I set up the bot?

Clone the repository, add a .env file with your DISCORD_TOKEN, CLIENT_ID, and GUILD_ID, run `npm install`, then `node index.js`.

Can I customize the commands?

Yes! Edit commands.js or implement a dynamic command loader for modular additions.

Does the bot support databases?

Economy commands use placeholders. Integrate SQLite or MongoDB for persistent data.
